Instrucciones de Uso

  1. En Modo Configuración puedes colocar piezas en el tablero usando el selector de piezas.
  2. En Modo Juego puedes controlar las piezas negras para crear problemas personalizados.
  3. Selecciona una pieza negra para ver sus movimientos válidos (marcados con puntos verdes).
  4. Haz clic en una casilla válida para mover la pieza seleccionada.
  5. Usa los botones para guardar y cargar tus problemas personalizados.
